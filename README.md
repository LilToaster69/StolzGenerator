# Stolzmonat Generator  
Erstelle ein #Stolzmonat Profilbild-Overlay  


## Installation

> NodeJs erforderlich

Repo klonen
`npm install` ausführen

Anwendung im Entwicklungsmodus starten
`npm run dev`


Anwendung erstellen
`npm run build`
